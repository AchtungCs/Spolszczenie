INSTALACJA:
1. Otwórz DBM'a
2. Kliknij w "Project"
3. Następnie wybierz opcje "Open Translations Directory"
4. Wrzuć plik "Polski.json" do tego folderu
5. Zrestartuj DBM'a
6. Po uruchomienu wejdź w opcję "Language" i wybierz język
7. Gotowe!
